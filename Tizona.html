This information (or parts of it) applies only to the console version of Terraria.

Tizona is currently the strongest sword in the Console version of the game and has the ability to cause bleeding on impact. It acts in the same way as Excalibur in that the player cannot turn while swinging, and has a base green/purple color scheme. When swung a yellow/red fire animation is shown, similar to meteor armor and the Fiery Greatsword. It attacks repeatedly when held down. It is a recommended sword for hardmode console players. The best possible prefix for this weapon is Legendary.

Trivia

  • Tizona is the sword carried by Rodrigo Daz de Vivar, El Cid(King's Champion), which was used to fight the Moors in Spain according to the Cantar del Mio Cid. The name Tizona translates to burning stick, firebrand.
  • It is a recolored version of the Excalibur.
