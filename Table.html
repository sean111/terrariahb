Table

A Table is a type of furniture item, commonly used in conjunction with Chairs, that can be placed on the ground. It is one of the items required for a housing to be considered valid.

Additionally, Tables can be used with chairs to create a miscellaneous crafting station, most notably used to craft Watches.

All tables are 3 blocks wide and 2 blocks tall.
