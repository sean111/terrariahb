This information (or parts of it) applies only to the PC version of Terraria.

The Diamond Robe is a robe that provides the wearer with 80 extra mana, and reduces the wearer's mana usage by 15%. It is the best gem-based robe in the game, and is an upgrade from the Ruby Robe.

Wearing the Wizard Hat alongside the Diamond Robe provides a set bonus of +10% magic critical strike chance. Any leg-slot equipment can be worn with the set bonus intact.
