The Trash Can is a storage item which has 40 storage slots. They essentially serve the same purpose as are Chests. Trash Cans can only be harvested when empty.The '"Trash Can'" will empty itself every 5 minutes making it useful for throwing things away.

Tips

  • The trash can requires four times as much iron/lead as a Chest. Therefore, unless used for the different aesthetic effect, chests or Barrels should be used instead.
  • The Trash Can can be used to differentiate relatively "useless" stored items from those of value.
  • If one is low on wood, and is no where near any trees, a trash can might be a preferable alternative.
Bug : All bugs that apply to Chests also apply to Trash Cans.

History

  • 1.1: Sell cost changed from 1 to 2 .
  • 1.0.6: Introduced.
