This information (or parts of it) applies only to the PC version of Terraria.

The Pumpkin Moon Medallion is used to summon the Pumpkin Moon. Like most boss-summoning items, it can only be used at night. Using a medallion merely converts your world's current night into a Pumpkin Moon - the night is not extended. This means that it is possible to "waste" a medallion by using it just before morning, which will automatically cancel out the Pumpkin Moon.
