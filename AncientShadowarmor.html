This information (or parts of it) applies only to the PC version of Terraria.

The Ancient Shadow Armor set is a throwback to the 1.0.3 Shadow Armor's look, but is otherwise identical in all ways to the current Shadow Armor, and can be interchanged with it. The ancient in the name is derived from the armor using an older version of the Shadow Armor's artwork.

Pieces of this set very rarely drop from Eater of Souls. They are also very rarely found in chests near the Corruption.

Notes

  • If you mix and match the ancient set and its current counterpart, you will still get the set bonus. However, you will not get the visual trailing shadow bonus that comes with the set.
  • The armor has the same sprite as Shadow Armor had before 1.0.3, thus the phrase "Ancient" in it.
  • If you have a decent amount of gear with a new player, you can farm Eater of Souls and skip to Shadow Armor very early on.
