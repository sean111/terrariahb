The Cursed Torch is a green colored version of the Torch. Unlike normal torches, the Cursed Torch can be held while underwater, and is placeable in water. It is crafted by combining one Cursed Flame with 33 torches. Using cursed torches for underwater exploring is usually better than using Glowsticks.
