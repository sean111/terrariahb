Adamantite Armor is a Hardmode armor, one tier below the (Hallowed Armor) and one tier above Mythril Armor. Like all Hardmode armors, it has three different headgear/helmet options, allowing the player to focus on magical, melee or ranged combat; each option changes the armor's set bonus. Equipping the full set gives off a faint pulsing aura around the character. Crafting the full set requires 78 Adamantite Bars, made from 390 Adamantite Ore.
