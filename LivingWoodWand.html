The Living Wood Wand is a Wand that was added in the 1.2 update. It places Living Wood blocks at the cost of 1 wood. It does not appear to use any Mana. The blocks placed are the same as found in Living Trees and have the same properties. It can be found in chests within the Underground Hallow structures and in some large living trees.
