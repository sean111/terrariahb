This information (or parts of it) applies only to the PC version of Terraria.

The Extractinator is an item that turns Silt and Slush into more valuable items like ores, coins, and gems. To use the Extractinator, blocks of silt or slush must be placed directly on the Extractinator as if it has been placed in the world. The blocks must be placed within the Extractinator's 33 item frame. By default, two blocks of material from a stack will be consumed per click. However, single blocks can be processed as well for the full output value.

It is a rare find in underground Gold Chests, and it is an occasional find in Ice Chests. Each Ice Chest has approximately a 5% chance of containing an Extractinator.

Notes

  • It is possible to obtain the alternates to the world-generated ores. If a world was generated with Gold Ore, the Extractinator can still produce Platinum Ore, which is otherwise unobtainable within that world.
  • Single blocks of silt and slush are accepted, but when using stacks, two blocks will be consumed at a time. To use single blocks in the Extractinator, a player must right-click on a stack of silt or slush to grab a single block from the stack. The player should then be able to left-click on the Extractinator to process a single block. While this method is much more efficient in terms of product created, it is much more time intensive than using a whole stack of material at once.
  • Using single blocks can be made easier through the use of mouse recording software while the game is running in a window.
  • The range of coin values obtainable from the Extractinator ranges from 1 Copper Coin to multiple Platinum Coins.
  • Feeding silt and slush to the Extractinator is the only way to obtain Amber and the Amber Mosquito.
