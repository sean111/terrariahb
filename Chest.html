The Chest is a Storage item, likely the first one the player will have access to. It can hold up to 40 item stacks in rows of 10x4. They can only be broken with a Pickaxe when empty, and it is impossible to destroy the blocks underneath the chest while it still has items in it.

In addition to being craftable, chests are also generated in caves close to the surface of the world, containing early game loot. They also spawn in the Dungeon, where they will always contain a Golden Key. Chests with loot spawn all over the rest of the world as well, but other biomes have different types of chest. In addition to the normal wooden chest, there are also Rich Mahogany Chests, Ebonwood Chests, Shadewood Chests and Pearlwood Chests, which must be crafted from their respective wood types and are never generated with the world.

Loot

The loot for any given chest is determined by the location in which it is found. Each chest contains a single "rare item" and an assortment of randomly chosen common items. The rare item is first placed in the chest, then each set of common items is checked to see whether or not it will be added. If a set is chosen to be added then a single item from the list is added to the chest.

Notes

  • Before opening a chest outside of a dungeon in a hardmode world, it would be advisable to throw something such as a boomerang or Magic Missile at it. If the chest is really a mimic, it will take damage and start chasing the player.
Bug 1.2.0.3.1: Placing a chest on a boulder and then making the boulder roll away will crash the game.
Bug 1.2: Chests placed on cobweb or diagonal ledges by world generation will duplicate several times if you place or destroy any block/torch next to it.
Bug 1.2: Chests that are generated on thin ice can duplicate.

Bug : Placing a chest next to a door with one block space and then opening the door into the chest will duplicate the chest while making the original chest "broken".
