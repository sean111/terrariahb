The Wooden Table is a Table made from Wood. Like all tables, it can be used as a flat surface item to create a valid NPC house, and, if paired with a Chair, a crafting station.

Many decorative items can be placed on the table. One of them is the Bottle, allowing a table to be used as an Alchemy Station.
