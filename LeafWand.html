This information (or parts of it) applies only to the PC version of Terraria.

The Leaf Wand is a Wand that was added in the 1.2 update. It places Leaf Blocks at the cost of 1 wood. It does not use any Mana. The blocks placed are the same as found in Living Trees and have the same properties. It can be found in a Living Wood Chest inside of a Living Tree.

History

  • 1.2: Introduced.
