Hallowed armor is a hardmode armor. Historically, it used to be the highest tiered armor from its introduction. It also used to be crafted from all the lower-down hardmode armors (Cobalt, Mythril and Adamantite, respectively), as well as 20 of each of the boss souls from the hardmode bosses of the time, one for each different piece. In the console version, the hallowed armor is one tier below, and an ingredient to craft, the Console exclusive armors.

With the 1.2 update, its crafting recipe has been simplified (now only requiring 54 Hallowed bars for a full set). Also, it has been superseded by a few new additional armor sets; the most comparable of which would be the Chlorophyte armor, which, like the Hallowed armor and many other hardmode armors, also has a ranged, melee and magic set.

This armor has both an after-image and pulsing aura visual effect; the same effect is used for the Cobalt and Adamantite armors.

Notes

  • Among all the existing armors:
    • The melee set provides the highest movement speed, melee speed and second highest critical strike chance (after Chlorophyte armor) bonuses.
    • The magic set provides the highest maximum mana (alongside Titanium armor), mana usage reduction and second critical strike chance (after Orichalcum armor) bonuses.
    • The ranged set provides the highest ammo reduction (alongside Adamantite armor) and second highest damage (after Titanium armor) bonuses.
